# SEO_checker

Требуется разработать систему проверки базовых значимых SEO-параметров сайта

Сценарий использования доступен по ссылке:
https://docs.google.com/document/d/1nY5jNIxp-SjkvP9VJdmnXXUlEt-Z8vpUhwtdP12N41o/edit?usp=sharing

На сайте должны быть страницы:
- Главная. Отображается список проверок (исторических)
- Детальная страница проверки (исторической)
- Модальное окно или страница ввода URL страниц, которые будем проверять;
- Страница с результатами текущей проверки.

Стек: PHP + Laravel, MySQL
